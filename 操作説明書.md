# VRChat Lottery Tool — 操作説明書

├───────────────────┬─────────────────────────────────────┤

│  返信予定時刻      │                                       │
└───────────────────┴─────────────────────────────────────┘

## ログイン画面

### 通常ログイン

1. ユーザー名（またはメールアドレス）を入力
2. パスワードを入力
3. 「ログイン情報を保存する」にチェックを入れると次回自動ログイン
4. **「ログイン」** ボタンを押す（または Enter キー）

### 2要素認証（2FA）が有効な場合

ログイン後に 2FA 入力画面に切り替わります。

- **TOTP**：Google Authenticator などの認証アプリの 6 桁コードを入力
- **メール OTP**：登録メールアドレスに届いた 6 桁コードを入力

「一つ前に戻る」リンクでログイン画面に戻れます。

### 自動ログイン

前回のセッション Cookie が有効な場合、起動時に自動でログインしてメイン画面が表示されます。

## メイン画面

### タイトルバー

| 要素 | 説明 |
|---|---|
| 現在時刻 | `yyyy/MM/dd HH:mm:ss` 形式で毎秒更新 |
| ユーザー名 @ ワールド名 | 自分の VRChat 表示名と現在のワールド名（30秒ごと更新）。ワールド名にカーソルを当てると完全な location 文字列が表示される |
| 状態バッジ | 現在のセッション状態（待機中 / 受付前 / 受付中 / 受付終了 / 完了など） |
| ログアウト | 確認ダイアログ後にログアウトしてアプリを終了 |

### 設定パネル

設定は変更するたびに `appsettings.json` に自動保存されます。

**当選人数**：「－」「＋」ボタンで 1〜20 人の範囲で変更できます。

**抽選モード**：トグルスイッチで「公平」「救済」を切り替えます。

**時刻設定**：`HH:mm` 形式で直接入力します（例：`21:00`）。

### 操作パネル

| ボタン | 有効な状態 | 動作 |
|---|---|---|
| セッション開始 | 待機中 | 設定に従ってセッションを作成・スケジュール登録 |
| セッション停止 | セッション進行中 | 確認後にセッションを中断・初期化 |
| 今すぐ抽選 | 受付終了後、応募者 1 人以上 | 手動で即座に抽選・Invite 送信 |
| テスト送信 | 常時 | 疑似 Request Invite を発生させて動作確認 |

### 応募者一覧

| 列 | 内容 |
|---|---|
| 表示名 | VRChat の表示名 |
| 応募時刻 | Request Invite を受信した時刻（HH:mm:ss） |
| ステータス | 「応募中」「🏆 当選」「無効」 |

当選者の行は背景色が変わります。

### ログパネル

色分けの凡例：

| 色 | 種別 |
|---|---|
| 白 | 情報（Info） |
| 緑 | 成功（Success） |
| 黄 | 警告（Warning） |
| 赤 | エラー（Error） |

「クリア」ボタンでログを消去できます。

## セッションの流れ

```
セッション開始
    │
    ▼
[受付前] ─── 受付開始時刻になったら
    │
    ▼
[受付中] ─── Request Invite を受信するたびに応募者一覧に追加
    │         ※ 同一ユーザーの重複は自動除外（最新 NotificationId に更新）
    │
    ▼
[受付終了] ─── 受付終了時刻になったら
    │
    ▼
[返信時刻] ─── 重み付き抽選を実行
    │            当選者に Invite を送信
    │
    ▼
[完了] ─── 2秒後に自動的に待機中へ戻る
```

---

## Invite 送信の仕組み

Request Invite の承認は以下の手順で行います。

1. `GET /users/{自分のID}` で自分の現在地（location）を取得
2. location から `worldId:instanceId` 形式の完全なインスタンス情報を取得
3. `POST /invite/{senderUserId}` に `instanceId` を添えて Invite を送信

**注意**：自分が VRChat のワールドに入室していない状態（offline / private / traveling）では Invite を送信できません。

## よくある質問

**Q. ログインしたのに「Pipeline 接続に失敗しました」と表示される**  
A. WebSocket の接続エラーです。テスト送信機能は使用可能です。少し時間をおいて再起動してみてください。

**Q. Invite が 400 エラーで失敗する**  
A. 自分が VRChat のワールドに入室していることを確認してください。ワールド名がタイトルバーに表示されている状態であれば正常です。

**Q. 設定が保存されない**  
A. `%AppData%\VRChatLotteryTool\appsettings.json` への書き込み権限を確認してください。

**Q. 履歴データを削除したい**  
A. `%AppData%\VRChatLotteryTool\lottery.db` を削除してください。ユーザーの当選履歴がリセットされます。

**Q. ログアウトしたい**  
A. タイトルバー右端の「🚪 ログアウト」ボタンを押してください。Cookie とセッション情報が削除されます。
