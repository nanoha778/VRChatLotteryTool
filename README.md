# VRChat Lottery Tool

VRChatの「Request Invite」を自動で受付し、抽選を行い、当選者へInviteを送信するWPFデスクトップアプリケーションです。

## 概要

本ツールは、複数人からのRequest Inviteを自動で管理し、指定ルールに基づいて抽選を行うことで、手動対応の手間を削減することを目的としています。

公平な抽選だけでなく、当選履歴に応じた「救済」ロジックにも対応しています。

## 主な機能

・Request Inviteの自動受付（WebSocket）
・重み付き抽選（公平 / 救済モード）
・指定時刻での自動Invite送信
・手動抽選（即時実行）
・応募履歴およびユーザー統計の保存
・自動ログイン（Cookie保存）

## 動作環境

・Windows 10 / 11
・.NET 10 Desktop Runtime
・VRChatアカウント（2FA対応）

## インストール

1. Releaseページからzipをダウンロード
2. 任意のフォルダに展開
3. VRChatLotteryToolSetup.exeを実行

データファイルは自動的にAppData配下に作成されます。

## 使い方

### 1. ログイン

起動後、VRChatのアカウントでログインします。
2要素認証が有効な場合はコード入力が必要です。

ログイン情報は保存可能で、次回以降は自動ログインされます。

### 2. セッション設定

・当選人数
・抽選モード（公平 / 救済）
・受付開始時間
・受付終了時間
・返信時間

を設定します。

### 3. セッション実行

「セッション開始」を押すと、以下が自動で実行されます：

・受付開始 → Request Invite受付
・受付終了 → 応募締切
・返信時間 → 抽選実行＋Invite送信

完了後は自動的に待機状態に戻ります。

また、「今すぐ抽選」で手動実行も可能です。

## 抽選モード

公平モード：
・ほぼ均等な確率で抽選
・直近当選者に軽いペナルティ

救済モード：
・連続落選者を優遇
・未当選ユーザーを優遇
・当選回数の多いユーザーを抑制

## データ保存先

%AppData%\VRChatLotteryTool\

・.authcookie（ログインセッション）
・.credentials（暗号化された認証情報）
・appsettings.json（設定）
・lottery.db（抽選履歴・統計）

## 注意事項

・VRChatのワールドに入室していない場合、Inviteは送信できません
・WebSocket接続が切れている間はRequest Inviteを受信できません
・テスト送信では実際のInviteは送信されません

## 既知の制限

・接続断中の通知は取得できません
・自動再送機能は未実装
